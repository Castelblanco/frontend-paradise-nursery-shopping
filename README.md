# Paradise Nursery Shopping

Frontend e-commerce

## Running Frontend

### Install Bun

<https://bun.com/docs/installation>

### Install Dependencies

Execute this command for install dependencies

```sh
bun i
```

### Run Frontend

Execute this command for run project

```sh
bun dev
```

### Build Frontend

Execute this command for create a build

```sh
bun run build
```

### Test Build in Preview

Execute this command for test the build as preview

```sh
bun run preview
```
